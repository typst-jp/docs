    module

    An evaluated module, either built-in or resulting from a file.

    You can access definitions from the module using field access notation and interact with it using the import and include syntaxes.

    Example

    #import "utils.typ"
#utils.add(2, 5)

#import utils: sub
#sub(1, 4)
