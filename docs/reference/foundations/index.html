    foundations

    基础类型和函数。

    在这里，您将找到有关基本数据类型（如 整数 和 字符串）以及核心计算功能的详细信息。

    定义

    • arguments捕获到函数的参数。
    • array一系列值。
    • assert确保条件得到满足。
    • bool具有两种状态的类型。
    • bytes字节序列。
    • calc用于计算和处理数字值的模块。
    • content文档内容的一部分。
    • datetime表示日期、时间或两者的组合。
    • dictionary从字符串键到值的映射。
    • duration表示正或负的时间跨度。
    • eval将字符串作为 Typst 代码进行评估。
    • float浮点数。
    • function从参数值到返回值的映射。
    • int整数。
    • label元素的标签。
    • module已评估的模块，可以是内置的或来自文件的结果。
    • panic失败并显示错误。
    • pluginWebAssembly 插件。
    • regex正则表达式。
    • repr返回值的字符串表示。
    • selector用于选择文档中的元素的过滤器。
    • strUnicode 代码点序列。
    • style提供对活动样式的访问。
    • sys用于系统交互的模块。
    • type描述一种值的类型。
    • version具有任意数量组件的版本。
