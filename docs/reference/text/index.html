    text

    文本样式。

    其中 文本函数 尤其有趣。

    定义

    • highlight高亮显示文本。
    • linebreak插入换行符。
    • lorem创建假文。
    • lower将字符串或内容转换为小写。
    • overline在文本上方添加一条线。
    • raw具有可选语法突出显示的原始文本。
    • smallcaps以小大写字母显示文本。
    • smartquote对其上下文做出反应的语言感知引号。
    • strike在文本中画一条线。
    • sub以下标形式呈现文本。
    • super以上标形式呈现文本。
    • text以各种方式自定义文本的外观和布局。
    • underline给文本加下划线。
    • upper将字符串或内容转换为大写。
